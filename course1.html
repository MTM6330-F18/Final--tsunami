Design Foundations

Teacher

Katie Tremblay

Topic's

  • Apply design styles, theories, and principles to digital graphics and layouts.
  • Use industry-standard software to create planning documents for various types of designs and formats.
  • Explain the purpose and functionality of the main tools and panels in industry-standard software.
  • Explain the effect that colour can have on a digital design.
  • Create digital documents that use and pair appropriate font families.
  • Use standard graphics software to design common web page layouts and the contained elements such as navigation menus, headers, and footers.
  • Use industry-standard software to create digital document layouts designed to be printed.
